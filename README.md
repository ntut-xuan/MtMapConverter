# MtMapConverter
將魔塔的地圖，使用機器學習來將地圖圖片轉成文字檔。



## Installation

1. clone 這個專案
2. `pip install -r requirements.txt`
3. Enjoy!



## File

DataSet 主要會放在 `image` 資料夾內，你可以自由的在裡面新增 77 x 77 的圖片進去。



## How to use?

1. 將魔塔地圖圖片 `mt.png` 放在專案根目錄上
2. `python main.py`
3. 可以得到 `Stage39_entity.txt` 與 `Stage39_material.txt`，即為地圖檔與物件檔 ID。
